Changelog

All notable changes to ZLZ Anime Shader are documented here.

Versions: 1.7.0 · 1.6.0 · 1.5.0 · 1.4.0 · 1.3.0 · 1.2.0 · 1.1.0 · 1.0.0

v1.7.0 — Receive Shadow Update

Released: June 15, 2026 · Current version

New Features

  • Added Soft Shadow support for Receive Shadow
  • Added Receive Shadow softness and sharpness controls
  • Added Self-Shadow Rejection support to prevent characters from receiving shadows cast by themselves while still receiving shadows from other objects
  • Added per-material Self-Shadow Rejection settings for greater customization

Improved Workflow

  • Improved Receive Shadow customization and control

Bug Fixes

  • Improved shadow rendering stability and character shadow separation

v1.6.0 — Shadow & Outline Update

New Features

  • Added Character Contact Shadow support for hair shadow replacement
  • Added Shadow Edge support for enhanced light and shadow transitions
  • Added Screen Space Outline support
  • Added Screen Space Outline Mask support

Improved Workflow

  • Updated WebGL Demo for v1.6.0

Bug Fixes

  • Fixed ShadowCaster support for proper Shadow Cascade rendering

v1.5.0 — Texture Mask & Character Effects Update

New Features

  • Added configurable Texture Mask Channel assignments for Shader features
  • Added Specular support for stylized highlights on characters and materials
  • Added Outline Selection support for character highlighting and target indication
  • Added Dither Camera Fade for automatic camera obstruction fading
  • Added Dither Occlusion Fade for automatic object occlusion fading
  • Added Texture Mask Packer — a tool designed to simplify texture mask creation and management:
    • Pack multiple mask textures into RGBA channels
    • Preview channel assignments before packing
    • Generate optimized mask textures for Shader features

Improved Workflow

  • Added runtime helper scripts to make FX features easier to set up and ready to use with minimal configuration

v1.4.0 — Character Workflow & Performance Optimization Update

New Features

  • Added Normal Map support
  • Added Alpha Channel support for masking unwanted Outline areas
  • Optimized Shader performance for lower rendering cost and better efficiency
  • Added Character Dashboard — a centralized hub designed to make character setup faster and more convenient:
    • Faster character setup with Auto Head Bone Detection
    • Smooth Normal Bake tool for higher-quality and more optimized Outline rendering
    • One-click Tone Mapping setup directly from the Character Dashboard
    • Auto Setup Outline Draws to reduce draw calls generated by Outline rendering
  • Added ZLZ Shader Optimizer — a tool designed to help analyze and optimize character performance more efficiently:
    • Overview of Materials, Shader Cost, and Texture Usage
    • Performance Budget presets for PC / Android / iOS
    • Toggle Shader features on/off to instantly preview Shader Cost impact
    • Compress Texture Size per Material or optimize the entire character at once

Bug Fixes

  • Fixed GI Lighting leaking into character shadows causing incorrect shadow colors
  • Fixed Filmic Curve incorrectly using Anime Mode Exposure settings

v1.3.0 — Stylized Tone Mapping & Workflow Update

This version introduces a new Tone Mapping system designed to enhance anime, cartoon, and stylized visuals. It also improves the usability of the Head Direction Binder script, making setup simpler and more intuitive for users.

What’s New

  • Added Auto-Detect Head Bone Axis, allowing one-click setup without manual configuration
  • Fixed an issue where Head Bone alignment was incorrect on certain characters
  • Introduced a Tone Mapping system for controlling brightness, contrast, and color intensity
  • Added two Tone Mapping modes: Anime and Filmic
  • Enabled full customization of Tone Mapping settings, with the ability to save and reuse presets
  • Added a Preview Curve in the Unity Editor for intuitive Tone Mapping adjustment

v1.2.0 — Face Shadow Accuracy & Workflow Update

This version focuses on improving the Face Shadow system and overall workflow, making it more accurate, easier to use, and visually refined. It also provides better control over behavior in the Editor, reducing unnecessary material changes and preventing unwanted Version Control updates.

What’s New

  • Improved facial lighting and shadow behavior to correctly respond to light direction
  • Enhanced accuracy of nose and facial shadow rendering
  • Smoother shadow fading, reducing harsh edges and abrupt cutoffs
  • Updated Face Shadow system to be driven by the Head Bone, improving compatibility with non-standard character orientations and overall accuracy
  • Added a helper to notify users if the required script or Head Bone for Face Shadow is not assigned
  • Updated Script: ZLZ_Head Direction Binder to support enabling/disabling Editor Preview:
    • Enabled: the script runs in the Editor, ensuring correct visual results immediately, but updates material values, which may cause changes in Version Control
    • Disabled: the script runs only at Runtime and does not send values to materials in the Editor, preventing changes in Version Control

v1.1.0 — Face Shadow Redesign

This update completely redesigns the Face Shadow system, making it more visually appealing, more flexible, and easier to use across a wide range of characters.

What’s New

  • Improved UX/UI to make the Face Shadow feature more prominent and easier to use
  • Supports a wide variety of face UV layouts
  • Includes a reusable Face Shadow texture (no need to switch textures per character)
  • Added Debug Face UV mode for setup and troubleshooting
  • Added UV Scale and Offset controls for precise adjustment
  • Added Flip UV option for handling inverted UVs

v1.0.0 — Initial Release

Released: March 3, 2026

Initial release of ZLZ Anime Shader for the Universal Render Pipeline (URP) — an anime-style character shader designed for real production use, suitable for both Artists and Technical Artists.

Features

  • Hair Transparent : Transparent hair rendering for clear facial visibility
  • Hair Shadow : Sharp and high-quality hair shadows that add depth and realism
  • Base Character Colors : Direct control over base color and shadow color of the character
  • Face Shadow : Face shadow control for clear and readable anime-style facial shading
  • Receive Realtime Shadow : Supports receiving realtime shadows from scene lighting
  • Lighting : Supports Main Directional Light and Additional Lights
  • Toon Shade Ramp : Controls sharpness and smooth transitions between light and shadow
  • Outline : Adjustable character outline with controllable thickness and color
  • Metallic : Tuned metallic surface support for anime-style materials
  • Hair Highlight : Adds stylized highlights to hair for shine and depth
  • Emissive : Controls emissive color and intensity for special effects
  • Dissolve : Dissolve effect for character appearance and disappearance
  • Transparency : Supports transparent materials for special use cases
  • Soft Light : Soft additive lighting to enhance skin smoothness
  • Rim Light : Rim lighting to separate the character from the background
  • Target Darken : Darkens the character to emphasize targets or important scenes
  • Indicator : Visual indicator effects for character states or targets
  • Get Hit : Reactive visual effect when the character is hit
  • Upgrade : Visual effects for character or equipment upgrades
  • Light Sweep : Sweeping light effect for showcasing weapons or special moments

Performance & Compatibility

  • Built for the Universal Render Pipeline (URP)
  • Optimized for mobile and real-time projects
  • Feature-based shader variants for efficient performance
  • Compatible with SRP Batcher
  • Designed for Mobile and PC platforms
