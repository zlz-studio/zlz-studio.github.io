# v1.5.0 — Texture Mask & Character Effects Update

### New Features

- Added configurable **Texture Mask Channel** assignments for Shader features
- Added **Specular** support for stylized highlights on characters and materials
- Added **Outline Selection** support for character highlighting and target indication
- Added **Dither Camera Fade** for automatic camera obstruction fading
- Added **Dither Occlusion Fade** for automatic object occlusion fading
- Added **Texture Mask Packer** — a tool designed to simplify texture mask creation and management:
  - Pack multiple mask textures into RGBA channels
  - Preview channel assignments before packing
  - Generate optimized mask textures for Shader features

### Improved Workflow

- Added runtime helper scripts to make FX features easier to set up and ready to use with minimal configuration
