# v1.7.0 — Receive Shadow Update

**Released:** June 15, 2026 · **Current version**

### New Features

- Added **Soft Shadow** support for Receive Shadow
- Added Receive Shadow **softness and sharpness controls**
- Added **Self-Shadow Rejection** support to prevent characters from receiving shadows cast by themselves while still receiving shadows from other objects
- Added **per-material Self-Shadow Rejection** settings for greater customization

### Improved Workflow

- Improved Receive Shadow customization and control

### Bug Fixes

- Improved shadow rendering stability and character shadow separation
