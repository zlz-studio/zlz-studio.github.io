### Parameter

- **Color (HDR)** : สีของ Specular
- **Intensity** : ความสว่างของ Specular ทั้งก้อน
- **Sharpness** : ควบคุมความแคบ/กว้างของ Highlight
- **Threshold** : บีบ Specular ให้แสดงผลในจุดที่สว่างที่สุด
- **Toon Step** : ความคมของ Specular
